# ExceLabel

ExceLabel 是一个面向 Windows 的本地桌面工具，用于扫描文件目录、导出 Excel 清单，并在 Excel 与本地文件系统之间执行双向同步。

## 当前实现状态

当前仓库已根据 [`plans/详细设计`](plans/详细设计) 建立第一版可运行工程，包含：

- Go 模块与基础目录结构
- 核心领域模型与改名冲突校验逻辑
- 并发文件扫描器
- 工作区状态持久化
- Excel 工作簿创建、读取与状态列回写
- 文件系统与工作簿监听器骨架
- 应用服务编排层
- Wails 绑定层入口
- React + Vite 前端界面，已接入真实 Wails 绑定调用
- 基础测试与开发说明

## 已完成的关键开发项

1. 已实现 Excel -> 文件系统 的重命名差异检测与状态回写，入口位于 [`SyncService.ApplyWorkbookChanges()`](internal/app/service.go:125)
2. 已补齐 文件系统 -> Excel 的增量对账基础流程，入口位于 [`SyncService.ApplyFilesystemChanges()`](internal/app/service.go:178)
3. 已实现基于逻辑路径与指纹的快照重建与记录复用，核心逻辑位于 [`reconcileEntries()`](internal/app/service.go:251)
